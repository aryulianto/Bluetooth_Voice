# Bluetooth_Voice
Program sederhana untuk menyalakan LED menggunakan Android melalui suara

# Requirement 
1. 1 Arduino Uno
2. 1 Module Bluetooth HC-05
3. 1 Breadboard
4. 4 Jumper Wire Female to Male
5. 4 Jumper Wire Male to Male
6. 1 LED
