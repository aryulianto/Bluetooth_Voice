# Bluetooth_Voice
Control your Arduino with voice commands using an Android smartphone! 

# Requirement 
- 1 Arduino Uno
- 1 Module Bluetooth HC-05
- 1 Breadboard
- 4 Jumper Wire Female to Male
- 4 Jumper Wire Male to Male
- 1 LED
